# Weather Report App
This is a responsive web application built with React that provides real-time weather information for any city worldwide. The app connects to the OpenWeather API to fetch live data including current temperature, "feels like" temperature, humidity, wind speed, and weather conditions.

# Tech Stack
React, JavaScript, CSS, OpenWeather API
